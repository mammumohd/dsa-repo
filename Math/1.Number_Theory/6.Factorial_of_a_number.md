# Factorial of a Number

Factorial of a number n = n * n-1 * n-2 ......1

# Iterative Solution

 ```
    function factorial(n) {
        var res=1;

        for (num=2; num<= n; num+++) {
            res = res * num;
        }

        return res;
    }

 ```

# Recursive 

 ```

    function factorial(n) {
        if( n == 0) {
            return 1;
        } 
        return n * factorial(n-1);
    }


 ```
